# AI Cyber Threat Intelligence Platform

An AI-powered Cyber Threat Intelligence Platform built using React, FastAPI, Machine Learning, and SQLite.

## Features

* Threat Severity Prediction
* Threat Analytics Dashboard
* User Authentication
* Prediction History
* MITRE ATT&CK Categories
* SOC Command Center
* PDF & CSV Export
* Live Threat Feed
* Risk Score Analysis

## Tech Stack

### Frontend

* React
* Recharts

### Backend

* FastAPI
* SQLAlchemy
* SQLite

### Machine Learning

* Scikit-learn
